Figure and figcaption

        
          Code snippet
          
          <figure>
            <img loading="lazy" alt="always include alt tag" src="./images/test.png"/>
            <figcaption>Title of image</figcaption> <!-- use css to, for example, make the caption appear on top of the image -->
          </figure>

Picture and source

          
            Code snippet 
            
                <picture>
                    <source srcset="./images/test.png"
                            media="(min-width: 800px)"> <!-- like an inline media query -->
                    <img loading="lazy" src="./images/Walking-your-dog-aggressive-Chihuahua-1.jpeg" alt="chi" />
                </picture>
